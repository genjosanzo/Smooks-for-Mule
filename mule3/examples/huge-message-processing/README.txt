+--------------------------------------------------+
| Smooks for Mule: huge message processing example |
+--------------------------------------------------+
This example demonstrates how Smooks with Smooks for Mule 3
handles huge gigabyte large data files and split, enrich, transform and
route the data to multiple endpoints.

This example is best run from its assembled form.
Run 'mvn clean install assembly:directory' to get the assembled
example in a directory form within the 'target' directory. Take
a look at it's README.txt file for more information.
